# BottleWebProject_С322_3_ EKP
## Оптимизационные задачи: ЗЛП, Транспортная задача и Задача о назначениях
### Описание

Данный репозиторий содержит программную реализацию, математические модели и готовые решения трёх классических задач математического программирования:

ЗЛП (Целочисленное линейное программирование)

Транспортная задача

Задача о назначениях

Все расчёты выполнены строго в соответствии с индивидуальным вариантом. 

## Технологии и зависимости

Язык: Python 3.9+

Обработка данных: pandas, numpy

Веб-фреймворк: Bottle

Среда разработки: Microsoft Visual Studio Community 2019/2022

Разметка и стили: HTML5, CSS3 (внешний файл /static/content/style.css)

Система контроля версий: Git + GitHub

## Постановка задач и методы решения

### 1. ЗЛП (Целочисленное линейное программирование)

Цель: Оптимизация линейной целевой функции при линейных ограничениях и условии целочисленности переменных.

Метод: Метод ветвей и границ / отсечения Гомори.

Данные: Вектор коэффициентов, матрица ограничений и правые части берутся из соответствующих разделов файла варианта.

### 2. Транспортная задача

Цель: Минимизация суммарной стоимости перевозок груза от поставщиков к потребителям.

Метод: Нахождение начального опорного плана (метод минимального элемента / северо-западного угла) + оптимизация методом потенциалов.

Данные: Векторы запасов и потребностей, а также матрица тарифов выгружаются из файла варианта. При несовпадении суммы запасов и потребностей задача автоматически приводится к закрытому виду добавлением фиктивного пункта.

### 3. Задача о назначениях

Цель: Распределение исполнителей по задачам так, чтобы минимизировать суммарные затраты.

Метод: Венгерский алгоритм.

Данные: Квадратная матрица стоимостей/эффективностей.
